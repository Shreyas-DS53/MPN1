Selenium Automation Framework – MPN1
📌 Project Description

This project is a Selenium Automation Framework developed using Java and TestNG.
It automates web application test scenarios with proper framework structure and reporting.

🛠 Tech Stack

Java

Selenium WebDriver

TestNG

Maven

Page Object Model (POM)

📂 Project Structure

src/test – Test scripts

Reports – Test execution reports

Screenshots – Failure screenshots

Logs – Execution logs

Master.xml – TestNG suite file

✅ Features

Structured framework design

Screenshot capture on failure

TestNG reporting

Reusable utilities

Clean folder organization

▶ How to Run the Project

Clone this repository

Open in Eclipse / IntelliJ

Update Maven dependencies

Run Master.xml as TestNG Suite

📊 Sample Output

Reports are generated inside the Reports folder after execution.
